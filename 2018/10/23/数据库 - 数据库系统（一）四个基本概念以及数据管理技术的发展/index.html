数据

  • 数据是什么(待更新)

    1. 数据（Data）是描述事物的符号记录，是指物理符号记录下来的，可以鉴别的信息，它的表现形式有多样，包括数字，字母，文字，特殊字符组成的文本数据，也可以是图形，图像，动画，影像，声音，语言等多媒体数据

      数据库系统原理（04735）P28

  • 信息

    1. 数据的解释是对数据含义的说明，也称为数据的语义，即数据所蕴含的信息
    2. 语义与其语义是密不可分，没有语义的数据是没有意义和不完整的
    3. 数据是信息存在的一种形式，只有通过解释或处理的数据才能成为有用的信息

      数据库系统原理（04735）P28

数据库

  • 数据库是什么

    1. 将大量数据保存起来，通过计算机加工而成的可以进行高效访问的数据集合

      《SQL基础教程》第二版

    2. 可以称为存储数据的仓库，这个仓库是存储在计算机存储设备上，并且所存储的数据是按一定的格式进行存储的

      数据库系统原理（04735）P28

    3. 严格意义上：数据库是指，长期储存在计算机中的有组织的，可共享的数据集合，且数据库中的数据按一定的数据模型组织，描述和存储，具有较小的冗余度，较高的数据独立性，系统易于扩展，并被多个用户共享

      数据库系统原理（04735）P28

  • 数据库的特点

    1. 永久存储
    2. 有组织
    3. 可共享

      数据库系统原理（04735）P28

数据库管理系统（Database Management System，DBMS）

  • 数据库管理系统（DBMS）是什么

    1. 用来管理数据库的计算机系统称为数据库管理系统（DBMS）

    2. 是专门用于建立和管理数据库的一套软件，介于应用程序和操作系统之间

      我们所用的 Dbeaver,Navicat Premium 是图形可视化管理工具，数据库管理系统就是存在它和操作系统之间的系统

    3. 功能：
      1. 数据定义功能
      2. 数据操作功能
      3. 数据库运行管理功能
      4. 数据库的建立和维护功能
      5. 数据组织，存储和管理功能

数据库系统原理（04735）P29

数据库系统

  • 数据库系统是什么
    1. 是指在计算机中引入数据库技术之后的系统
    2. 通常，一个完整的数据库系统包括数据库，数据库管理系统以及相关实用工具，应用程序，数据库管理员和用户
  • 数据库管理员（Database Administratior, DBA）
    1. 专门对维护数据库进行维护，保证数据库正常，高效运作的人员

  • 数据库系统的用户

    1. 应用的用户
    2. 开发应用的程序员

DBMS 的种类

  1. 层次数据库(Hierarchial Database, HDB)

  2. 关系数据库(Relational Database, RDB)

    • 这种类型的 DBMS 称为关系型数据库管理系统(Relational Database Management System, RDBMS)
    • 代表性的 RDBMS
      1. Oracle Database: 甲骨文公司的 RDBMS
      2. SQL Server: 微软公司的 RDMBS
      3. DB2: IBM 公司的 RDBMS
      4. PostgreSQL: 开源的 RDBMS
      5. MySQL: 开源的 RDBMS

  1. 面向对象数据库(Object Oriented Database, OODB)

  2. XML 数据库(XML Database, XMLDN)

  3. 键值存储系统（key-Value Store, KVS）
