什么是 PHP

  • PHP 是一种专门为 WEB 设计的服务器脚本语言

PHP 的诞生和发展

  • 1994年出现 Rasmus Lerdorf
  • PHP 最初的含义：Personal Home Page(个人主页)，后来使用 GNU 命名规则（GNU = Gnu’s Not UNIX）如今是 PHP 超文本预处理程序（PHP, Hypertext Preprocessor）的缩写
  • PHP 如今的版本：7，主页：http://www.php.net。PHP 7 的引擎 Zend 经过完全重写，其主页：http://www.zend.com
