在 HTML 文档中嵌入 PHP

请注意：这个文件的后缀是 .php 请不要使用 .html 结尾，不然会把 PHP 中 ?> 现在页面中

为什么会这样呢？

  1. 浏览器并不理解 PHP 代码，所以在以 .html 结尾的资源中嵌入 PHP 代码，浏览器会把 PHP 当作 HTML 文档来解释

  2. 若是采用 .php 为后缀的资源，我们在访问该资源的时候。PHP 代码已经在服务端被 PHP 的解释器处理过
