PHP - 基础知识（三）使用操作符

发布日期 2018-11-05 | In 书籍笔记 , 《PHP and MySQL Web Development》第五版

操作符初步认识，后续继续更新（2018-11-05）

代数操作

  1. 通常是整型和双精度的数据操作
  2. 如果应用于字符串，PHP 会试图将这些字符串转化为数字（和 JavaScript 的操作类似）
  3. 如果包含 e 和 E 则会转化为科学计数法变成浮点数，否则会当作整数进行处理

赋值操作符

  1. 赋值操作符，不能简单的理解为等于
  2. 规则是： 整个赋值语句的值将赋给左边的操作数
  3. 使用圆括号来提供表达式的优先级（同数学中的计算法则类似）

复合赋值操作符

  1. += 和 -=
  2. 前置递增递减（–$varname，++$varname）
  3. 后置递增递减（$varname–，$varname++）

位操作符

  1. & 按位与
  2. | 按位或
  3. ~ 按位非
  4. ^ 按位异或
  5. << 左位移

  6. 右位移

比较操作符

  1. 等于操作符 == 值相等就行
  2. 恒等于操作符 === 数据类型和值都要相等
  3. 不等 !=
  4. 恒不等 !=
  5. 不等操作符 <>
  6. 小于操作符 <
  7. 大于操作符 >
  8. 小于操作符 <=
  9. 大于操作符 >=

逻辑操作符

  1. ! (NOT)
  2. && (AND)
  3. || (OR)
  4. and (AND)
  5. or (OR)
  6. xor (XOR)

    后三个逻辑操作符要比前三个逻辑操作符的优先程度要低

引用操作符

  1. & 操作符

    区别非引用的情况

字符串操作符

  1. 字符串连接（.）：将两个字符串连接起来生成并保存到一个新字符串中

其他操作符

  1. , 逗号操作符 ，用于分隔函数参数和其他列表项
  2. ?: 三元操作符，类似于条件语句 if else
  3. @ 错误抑制符，可以放在任何表达式前面使用
  4. 执行操作符，PHP 会试着将反向单引号中的值当作服务器端命令行命令来执行
  5. []（=>） 数组操作符，支持访问数组元素
