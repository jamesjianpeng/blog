Q1: 需要把 SQL 语句包装成 Promise 对象，不然访问指定的 api 会返回 404

A: 在我们访问 指定的 API 的时候，会执行查询或更新等操作，这个操作是异步的，若不用包装成 Promise 对象，没有的等前面的操作完成 Koa 就直接响应客户端

Q2: 返回值乱码的问题，

A: 需要在 content-type 后面加上 charset=utf-8
