2. 更新表
  • 目的：为了实现数据中表规范化设计的目的，有时需要对之前已经创建的表做进一步的结构修改和调整。
  • 在 MySQL 中可以使用 ALTER TABLE 语句来更改原有表的结构，如：
    • 增加或删减列
    • 创建或取消索引
    • 更改原有列的数据类型
    • 重新命名列或表
    • 更改表的评注和表的引擎类型等

  • ADD [ COLUMN ] 子句: 可添加新的一列

  • 实践：

  • 创建一个表之后发现少了一个属性：年龄，下面是对 students 表的更新操作

  • CHANGE [ COLUMN ] 子句：改变一列，需要严格按照书写顺序（也可以修改多列，他们之间使用 逗号隔开）

    • ⚠️请注意 CHANGE COLUMN 后面的顺序：旧的属性名 新的属性名 新的数据类型 新的约束 指定默认值等

    • 单列修改

      mysql> ALTER TABLE table_name

  -> old_column_name new_column_name data_type> value;
