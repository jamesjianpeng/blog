属性选择器

  • 属性选择器出现的背景：类选择器和 ID 选择器可以在 HTML / SVG / MathML 文档中使用，但是在 其他标记语言中是不能使用，所以 CSS2 就引入属性选择器
  • 作用：可以根据元素的属性以及属性值来选择元素，一共有 4 种类型的属性选择

  • 4 种分类：1. 简单属性选择。2. 根据具体属性值选择。3. 根据部分属性值选择。 4. 特定属性选择类型。

特殊性

规则：

  • !important 的优先级最高（大于内嵌等其他的选择器组合）例子
  • 内嵌的样式优先级 > style 包裹的 > link 外链
  • 对于选择器中给定的各个 ID 属性值，加 0.1.0.0
  • 对于选择器中给定的各个类属性值，属性选择或伪类，加 0.0.1.0
  • 对于选择器给定的各个元素和伪元素，加 0.0.0.1
  • 结合符和通配选择器对特殊性没有任何贡献
  • 当优先级相同的时候，在相同区域内，后者覆盖前者的样式
  • 比较的时候从左向右进行对比
