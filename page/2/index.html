API 预习：

-通过 draggable 定义可拖拽的目标

  • 可拖动目标的是事件: dragstart drag dragend
    • dragstart：拖拽元素的开始的时候触发（在拖拽元素的一个生命周期只会触发一次）
    • drag：拖拽元素在拖拽的时候会反复触发（在拖拽元素的一个生命周期会反复触发）
    • dragend：拖拽元素的被鼠标释放的时候（在拖拽元素的一个生命周期只会触发一次）

javascript

  • 定义释放区
    • 通过定义 dragenter, dragover, dragleave, drop
      • dragenter：拖拽元素进入释放区的时候，会被触发（在释放区的一个生命周期中可能会被反复触发）
      • dragover：拖拽元素停留（移动）在释放区的时候，会被触发（在释放区的一个生命周期中会被反复触发）
      • dragleave：拖拽元素离开释放区的时候，会被触发（在释放区的一个生命周期中可能会被反复触发）
      • drop：拖拽元素移动到释放区并被鼠标释放，会触发（在释放区的一个生命周期中只会触发一次）

查询操作

地位：数据查询是 SQL 语言的核心功能，也是数据库中使用最多的操作
作用：是从数据库中的一张或多张表（或视图）中检索出满足条件的数据信息

通常，查询的操作是由 0 行（没有满足条件的数据）或多行记录组成的一个记录集合，并允许选择一个或多个字段作为输出字段

数据理论基础：是关系数据模型中对表对象的一组关系，即选择，投影和连接

特殊性

规则：

  • !important 的优先级最高（大于内嵌等其他的选择器组合）例子
  • 内嵌的样式优先级 > style 包裹的 > link 外链
  • 对于选择器中给定的各个 ID 属性值，加 0.1.0.0
  • 对于选择器中给定的各个类属性值，属性选择或伪类，加 0.0.1.0
  • 对于选择器给定的各个元素和伪元素，加 0.0.0.1
  • 结合符和通配选择器对特殊性没有任何贡献
  • 当优先级相同的时候，在相同区域内，后者覆盖前者的样式
  • 比较的时候从左向右进行对比
