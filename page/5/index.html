聚合

  • 聚合：就是将多行汇总成一行
  • 聚合的方式：
    • 使用聚合函数（COUNT, SUM, AVG …）
    • 使用分组子句（GROUP BY）
    • 使用子句(HAVING)

聚合函数

  • 聚合函数（聚集函数）：用于汇总的函数

    COUNT: 计算表中的记录条数（行数）

GROUP BY 子句

  • 在 GUROUP BY 子句中指定的列称为聚合键或者分组列

  • 和 SELECT 结合使用，子句的顺序（不能改变也不能替换）：

      1. SELECT – FROM – WHRER – GROUP BY

    • 分组会也会把 NULL 挑选出来为一新列

####

  • SELECT – FROM – GROUP BY 例子：

    将 purchase_price 作为 聚合键（分组列），展示出它们的总数

  • SELECT – FROM – WHRER – GROUP BY 例子：

    有 WHERE 子句先过滤掉对应的数据
    在将 purchase_price 作为 聚合键（分组列），展示出它们的 product_type 和 COUNT 的结果

使用 GROUP BY 时注意的地方

  • 使用 GROUP BY 子句时，SELECT 子句中不能出现聚合键（分组列）之外的列名
  • 不能使用 alias（别名）
  • 在 SELECT 子句中没有指明排序的方式，数据的展示时无序的
  • 在 WHERE 子句中使用 聚合函数

SQL 初识

发布日期 2018-10-14 | In 书籍笔记 , 《SQL基础教程》第二版

数据库

什么是数据库

  • 是一个以某种有组织的方式存储的数据集合

数据库其他的要素

  • 表（table）：一个数据库中可以存在多个表，比如：图书馆中的书籍分类：人文历史 和 青春文学 应该是不同的表
  • 列（column）-> 数据类型：表中的一个字段
  • 行（row）：表中的一条记录
  • 单元格：列和行的交汇处；一个单元格只能输出一个数据
  • 主键（primary key）：用于区分一行的标识，可以通过主键把该条数据找出来

SQL

什么是 SQL

  • 是结构化查询语言（Structured Query Language）
  • 是一种专门用来与数据库通信的语言
  • 用 关键字，表名，列名等组合而成的一条语句（ SQL 语句）来描述操作内容

SQL 的基本书写规则

  • 语句要以分号（ ; ）或者 /g 结尾
  • 关键字，不区分大小写
  • 命名规则：只能使用半角英文字母，数字，下划线（_）,作为数据库，表和列的名字
  • 表名和列名不能重复
  • 使用中文的时候请用双引号包裹

  • 关于注释：/ xxx /（多行注释） 或者 –（一行注释）

    或者

    使用 – 注释的时候注意需要换行

SQL 语句的种类（90% 都是 DML）

  • DDL (Data Definition Language，数据定义语言)

    CREATE: 创建数据库和表等对象
    DROP: 删除数据库和表等对象
    ALTER: 修改数据库和表等对象的结构

  • DML (Data Manipulation Language，数据操纵语言)

    SELECT: 查询表中的数据
    INSERT: 向表中插入新数据
    UPDATE: 更新表中的数据
    DELETE: 删除表中的数据

  • DCL (Data Control Language, 数据控制语言)

    COMMITL: 确认对数据库中的数据进行的变更
    ROLLBACK: 取消对数据库中的数据进行的变更
    GRANT: 赋予用户操作权限
    REVOKE: 取消用户的操作权限

  • 事务控制语言

希尔排序：

  • 为什么希尔排序会选择插入排序进行优化？？(暂时没有领悟到😂)

  • 特点：

    • 这是属于插入排序的升级版
    • 插入排序和希尔排序的最大区别：对比的间隔有区别
      • 插入排序的间隔是不变的：都是1，每一与相邻的元素进行比较
      • 希尔排序则不同：每次对比的间隔是可能不会相同，往往先对比远距离的元素，然后对比的距离慢慢缩小

  • 排序效率：(使用执行时间对比，和最深层循环的计数器比较的，dome 地址)

    • 选择排序 ~= 冒泡排序 < 插入排序 < 希尔排序

  • 在上述几种排序中希尔排序效率高？（我的理解不透彻，具体原因不知，待更新）

    • 影响排序效率的因素
      1. 最深层循环的总次数
      2. 每次循环做的事情
