-----------------
### CSS 分类

1. 透明：
    CSS3: opacity, rgba()

2. 圆角：
    CSS3: border-radius

3. 背景：
    CSS3: background-image, background-position, background-repeat

4. 阴影(CSS3)

  * 盒子阴影
      CSS3: box-shadow [IE 9]

  * 文本阴影
      CSS3: text-shadow [IE 10]

5. 渐变(CSS3)
  * 线性渐变: linear-gradient() [IE 10]
  * 放射性渐变: radial-gradient() [IE 10]
